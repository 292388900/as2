Architecture

The as2 server contains of several components that allow to communicate with both the trading partners and the internal integratino system.


AS2 Server:
The server is the core component. It is responsible for the transaction processing and care for encryption, digital signatures and the communication to all the other components.

AS2 Client:
The AS2 client contains the transaction management, partner management, certificate management. It allows to set all server properties.

DB Server:
The database server stores all information about the transactions and the master data of partners, subjects etc.


HTTP Server:
The HTTP server acts as a servlet container for the message receipt servlet. It host alos some information pages. The servlet sends received messages via RMI to the server.

 

HTTP Sender:
This component sends messages and MDN (async) to the trading partner. It also received MDN if the MDN of an outbound transmission is set to sync.
